# Kairos v1.1.6

## Highlights

- Simplified the VS Code status bar to show coding time only with an icon-first label.
- Reworked hover details to focus on the current coding session instead of transport state.

## VS Code Extension UX

- Status bar text now uses `$(code)` and displays only tracked coding time.
- Removed buffering, idle, and connection wording from the status bar and hover.
- Hover now emphasizes session-focused details:
  - today tracked time
  - current session duration
  - live/paused session state
  - session start and last activity timestamps
  - active file and language context

## Quality

- Updated runtime snapshot fields to include current session metadata.
- Added and updated status bar unit tests for icon-first time display and session hover content.
