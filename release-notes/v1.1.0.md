# Kairos v1.1.0

## Highlights

- Improved workspace attribution consistency across the VS Code extension and desktop analytics.
- Added canonical handling for no-workspace contexts so historical and current sentinel values are merged.
- Polished project labeling in desktop UI by showing a user-friendly `No workspace` display label.
- Cleaned and reorganized repository README with user-facing guidance first and developer workflows grouped at the bottom.

## macOS Install Guidance

- Added clear Gatekeeper bypass steps in the README for unsigned builds.
- Documented why macOS can block launch for unsigned/notarized-missing builds.
- Added update-time troubleshooting guidance when Gatekeeper prompts again after upgrades.

## Notes

- This release keeps the local-first architecture intact with no cloud dependency changes.
