# Kairos v1.1.2

## Fixes

- Fixed notification panel styling to correctly match dark theme surfaces and text colors.
- Removed semi-transparent/glass toast backgrounds that reduced readability.
- Standardized in-app notification and toast appearance using theme tokens for consistent contrast.

## User Impact

- Notification dropdown now looks native in dark mode.
- Toast messages are fully legible and no longer washed out by transparency.
