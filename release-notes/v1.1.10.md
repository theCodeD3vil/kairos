# Kairos v1.1.10

## Highlights

- Fixed the macOS menubar popover so it now closes reliably when clicking outside.
- Improved menubar interaction handling with explicit outside-click monitoring to avoid sticky popovers.

## Desktop App

- Added local and global click monitors around the menubar popover lifecycle.
- Outside clicks now close the popover unless the click lands on the popover itself or the menubar status button.
- Cleaned up popover monitor teardown on close/dealloc to prevent stale event monitors.
