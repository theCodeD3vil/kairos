# Kairos v1.1.5

## Highlights

- Stabilized VS Code status bar state transitions to reduce rapid buffering/reconnect flicker.
- Improved desktop bridge resilience against transient websocket timeout and probe failures.

## VS Code Extension Reliability

- Increased desktop websocket request timeout from `1.5s` to `5s` to reduce false timeouts on busy machines.
- On websocket request timeout, extension now resets the stale socket immediately so the next request reconnects cleanly.
- Added connection probe hysteresis so a single probe miss does not instantly downgrade the status bar state.
- Added replay failure smoothing so a single transient ingest timeout no longer forces immediate buffering state.

## Verification

- Added targeted runtime/client tests covering timeout recovery and probe/replay transient behavior.
- Confirmed extension test suite and typecheck pass with the new transport hardening.
